Error
In: ruby-1.8.7-p22/ext/zlib/zlib.c Parent:  StandardError

The superclass for all exceptions raised by Ruby/zlib.

The following exceptions are defined as subclasses of Zlib::Error. These exceptions are raised when zlib library functions return with an error status.

  • Zlib::StreamEnd
  • Zlib::NeedDict
  • Zlib::DataError
  • Zlib::StreamError
  • Zlib::MemError
  • Zlib::BufError
  • Zlib::VersionError
