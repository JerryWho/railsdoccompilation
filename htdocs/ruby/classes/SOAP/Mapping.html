name2elename(name)

Allow only (Letter | ‘_’) (Letter | Digit | ’-’ | ‘_’)* here. Caution: ’.’ is not allowed here. To follow XML spec., it should be NCName. 

  (denied chars) => .[0-F][0-F]
  ex. a.b => a.2eb

191:     }.gsub(/::/n, '..')
