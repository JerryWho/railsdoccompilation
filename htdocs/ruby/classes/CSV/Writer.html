Writer
In: ruby-1.8.7-p22/lib/csv.rb Parent:  Object

CSV formatted string/stream writer.

EXAMPLE 

  Write rows to 'csvout' file.

  outfile = File.open('csvout', 'wb')
  CSV::Writer.generate(outfile) do |csv|
    csv << ['c1', nil, '', '"', "\r\n", 'c2']
    ...
  end

  outfile.close
