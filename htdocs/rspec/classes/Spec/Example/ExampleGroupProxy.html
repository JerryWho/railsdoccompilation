ExampleGroupProxy
In: lib/spec/example/example_group_proxy.rb Parent:  Object

Lightweight representation of an example group. This is the object that is passed to Spec::Runner::Formatter::BaseFormatter#add_example_group

Methods
  • backtrace
  • filtered_description
Attributes
[R] description This is the docstring passed to the describe() method or any of its aliases
[R] examples A collection of ExampleGroupProxy instances, one for each example declared in this group.
[R] location The file and line number at which the represented example group was declared. This is extracted from caller, and is therefore formatted as an individual line in a backtrace.
[R] nested_descriptions Used by Spec::Runner::Formatter::NestedTextFormatter to access the docstrings for each example group in a nested group.
