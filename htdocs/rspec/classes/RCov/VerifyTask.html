Class
VerifyTask
In: lib/spec/rake/verify_rcov.rb Parent:  Rake::TaskLib

A task that can verify that the RCov coverage doesn’t drop below a certain threshold. It should be run after running Spec::Rake::SpecTask.

Methods
  • define
  • new
Attributes
[RW] index_html Path to the index.html file generated by RCov, which is the file containing the total coverage. Defaults to ‘coverage/index.html‘
[RW] name Name of the task. Defaults to :verify_rcov
[RW] require_exact_threshold Require the threshold value be met exactly. This is the default.
[RW] threshold The threshold value (in percent) for coverage. If the actual coverage is not equal to this value, the task will raise an exception.
[RW] verbose Whether or not to output details. Defaults to true.
