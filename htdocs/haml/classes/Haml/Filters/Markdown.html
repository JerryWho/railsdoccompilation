Module
Markdown
In: lib/haml/filters.rb

Parses the filtered text with [Markdown](http://daringfireball.net/projects/markdown). Only works if [RDiscount](http://github.com/rtomayko/rdiscount), [RPeg-Markdown](http://github.com/rtomayko/rpeg-markdown), [Maruku](http://maruku.rubyforge.org), or [BlueCloth](www.deveiate.org/projects/BlueCloth) are installed.
