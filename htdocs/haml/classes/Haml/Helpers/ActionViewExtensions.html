page_class()

Returns a value for the "class" attribute unique to this controller/action pair. This can be used to target styles specifically at this action or controller. For example, if the current action were `EntryController#show`, 

    %div{:class => page_class} My Div

would become 

    <div class="entry show">My Div</div>

Then, in a stylesheet (shown here as {Sass}), you could refer to this specific action: 

    .entry.show
      font-weight: bold

or to all actions in the entry controller: 

    .entry
      color: #00f

@return [String] The class name for the current page

This method is also aliased as generate_content_class_names
