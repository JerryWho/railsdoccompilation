HTML
In: lib/haml/html.rb Parent:  Object

Converts HTML documents into Haml templates. Depends on [Hpricot](http://github.com/whymirror/hpricot) for HTML parsing.

Example usage: 

    Haml::Engine.new("<a href='http://google.com'>Blat</a>").render
      #=> "%a{:href => 'http://google.com'} Blat"
