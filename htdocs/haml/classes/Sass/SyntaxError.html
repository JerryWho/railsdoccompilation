Class
SyntaxError
In: lib/sass/error.rb Parent:  StandardError

An exception class that keeps track of the line of the Sass template it was raised on and the Sass file that was being parsed (if applicable).

All Sass errors are raised as {Sass::SyntaxError}s.

Attributes
[R] sass_filename The name of the file that was being parsed when the exception was raised. This could be `nil` if no filename is available.

@return [String]

[RW] sass_line The line of the Sass template on which the error occurred.
