Attributes
[R] members The array of {SimpleSequence simple selector sequences}, operators, and newlines. The operators are strings such as `"+"` and `">"` representing the corresponding CSS operators. Newlines are also newline strings; these aren’t semantically relevant, but they do affect formatting.

@return [Array<SimpleSequence, String>]

superselector?(sseq)

Returns whether or not this selector matches all elements that the given selector matches (as well as possibly more).

@example (.foo).superselector?(.foo.bar) #=> true (.foo).superselector?(.bar) #=> false (.bar .foo).superselector?(.foo) #=> false

@param sseq [SimpleSequence] @return [Boolean]
