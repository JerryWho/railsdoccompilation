Module
RX
In: lib/sass/scss/rx.rb

A module containing regular expressions used for lexing tokens in an SCSS document. Most of these are taken from [the CSS3 spec](http://www.w3.org/TR/css3-syntax/#lexical), although some have been modified for various reasons.
