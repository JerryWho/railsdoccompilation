The abstract superclass for SassScript parse tree nodes.

Use \{perform} to evaluate a parse tree.

Attributes
[R] context The context in which this node was parsed, which determines how some operations are performed.

Can be `:equals`, which means it’s part of a `$var = val` or `prop = val` assignment, or `:default`, which means it’s anywhere else (including `$var: val` and `prop: val` assignments, `#{}`-interpolations, and other script contexts such as `@if` conditions).

@return [Symbol]

[RW] line The line of the document on which this node appeared.

@return [Fixnum]

[R] options The options hash for this node.

@return [{Symbol => Object}]
