Overview

Raised when number of bind variables in statement given to :condition key (for example, when using find method) does not match number of expected variables.

For example, in 

  Location.find :all, :conditions => ["lat = ? AND lng = ?", 53.7362]

two placeholders are given but only one variable to fill them.
