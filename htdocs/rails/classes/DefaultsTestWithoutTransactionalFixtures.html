test_mysql_integer_not_null_defaults()

MySQL uses an implicit default 0 rather than NULL unless in strict mode. We use an implicit NULL so schema.rb is compatible with other databases.

test_mysql_text_not_null_defaults()

MySQL 5 and higher is quirky with not null text/blob columns. With MySQL Text/blob columns cannot have defaults. If the column is not null MySQL will report that the column has a null default but it behaves as though the column had a default of ’’
