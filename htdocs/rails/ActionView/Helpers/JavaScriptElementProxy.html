- (Object) [](attribute)

Allows access of element attributes through attribute. Examples: 

  page['foo']['style']                  # => $('foo').style;
  page['foo']['style']['color']         # => $('blank_slate').style.color;
  page['foo']['style']['color'] = 'red' # => $('blank_slate').style.color = 'red';
  page['foo']['style'].color = 'red'    # => $('blank_slate').style.color = 'red';
