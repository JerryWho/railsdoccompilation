Module
Screenshots
In: examples/watir/features/support/screenshots.rb

This is an example of how you can set up screenshots for your browser testing. Just run cucumber with —format html —out report.html

The code below will work on OS X or Windows (with IE Watir only). Adding support for other platforms should be easy - as long as there is a ruby library or command line tool to take pictures.
