Classes in this module parse feature files and translate the parse tree (concrete syntax tree) into an abstract syntax tree (AST) using <a href="Embedded" target="_blank">http://martinfowler.com/dslwip/EmbeddedTranslation.html">Embedded translation</a>.

The AST is built by the various build methods in the parse tree.

The AST classes are defined in the Cucumber::Ast module.
