Configuration
In: lib/webrat/core/configuration.rb Parent:  Object

Webrat can be configured using the Webrat.configure method. For example: 

  Webrat.configure do |config|
    config.mode = :sinatra
  end

Attributes

[RW] application_environment Which rails environment should the selenium tests be run in? Defaults to test.
